# OPChat
适用于管理员线上沟通的聊天室（非聊天室内的玩家无法看到消息）

## 插件功能
- 拥有权限的玩家可以进入聊天室进行聊天
- 非聊天室内的玩家无法看到聊天室内的消息
- 聊天室内的玩家可以看到公屏消息

## 指令＆权限
- /oc toggle - 切换聊天模式
- /oc help - 查询帮助
- /oc reload - 重载插件（将所有人踢出聊天室，用来修复偶然产生的“复读机”bug）
**权限节点：**opchat.all
